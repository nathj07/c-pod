Systems Team Repository

This server hosts all things used in configuring our OS and application environments:
  • DVD Images for the operating systems we use
  • Downloads for various things
  • Kickstart Configurations
  • GEM Repository
  • YUM Repositories
    • Stable RPMs with our customizations
    • Unstable RPMs under testing
    • RPM Source Packages
  • CentOS
    • CentOS Package Repository for network installs
    • CentOS Distribution DVD from the Distribution DVD
Note that our current standard build of Centos is 5.9 but 6.4 is being tested now...

Downloads

There are other files that we rely on. For example:
  • centosboot.iso - A minimal boot image (~10Mb). Download this and boot from it to obtain a prompt from which you can choose different types of install. For a networked kickstart install type linux ks=http:///downloads/centos-vm.ks to perform a VM install. This file is symlinked to the current Custom version specific file.
  • The latest copy of VMware tools
  • Git for Mac

GEM Repository

We have created some of our own GEMS, so a GEM repository has been setup. To use this add the source option to your GEM installs: 
gem install --source http:///gem_repo
Or you can download them directly and install locally.

YUM Repositories

Custom RPMs (Stable)

In order to use this repository you will need to create the file /etc/yum.repos.d/custom.repo with the following contents: 

[custom]
name=Custom RPMs for Centos $releasever - $basearch
baseurl=http:///repos/custom/$releasever/stable/
enabled=1
gpgcheck=0

Browse the Custom RPMs

Unstable Custom RPMs for Testing

When developing a new release of a custom RPM this repository is used. This is because YUM repos don't readily support multiple versions.

In order to use this repository you will need to create the file /etc/yum.repos.d/custom_unstable.repo with the following contents: 

[custom_unstable]
name=Unstable RPMs for Centos $releasever - $basearch
baseurl=http:///repos/custom/$releasever/unstable/
enabled=1
gpgcheck=0
Browse the Unstable Custom RPMs

CentOS Package Repository

In order to use this repository you will need to create the file /etc/yum.repos.d/centosdvd.repo with the following contents: 

[centos_repo]
name=CentOS Enterprise Linux Distribution DVD - x86_64
baseurl=http:///repos/centos/$releasever/repo/
enabled=1
gpgcheck=1
gpgkey=file:///etc/pki/rpm-gpg/RPM-GPG-KEY-CentOS-5
exclude=postgresql*

Browse the Distribution DVD Repository
