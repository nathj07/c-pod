Systems Team Repository

This server hosts all things used in configuring our OS and application environments:
  • DVD Images for the operating systems we use
  • Downloads for various things
  • Kickstart Configurations
  • GEM Repository
  • Stable RPMs with our customizations
  • Unstable RPMs under testing
  • CentOS Package Repository from the Distribution DVD
  • CentOS Distribution DVD for network installs
  • RPM Package Sources
Note that our current standard build of Centos is 5.8 but 6.2 is being tested now...

Downloads

There are other files that we rely on. For example:
  • centosboot.iso - A minimal boot image (~10Mb). Download this and boot from it to obtain a prompt from which you can choose different types of install. For a networked kickstart install type linux ks=http://repo-st.oak.iparadigms.com/downloads/centos-vm.ks to perform a VM install. This file is symlinked to the current iParadigms SYE version specific file.
  • The latest copy of VMware tools
  • Git for Mac

GEM Repository

We have created some of our own GEMS, so a GEM repository has been setup. To use this add the source option to your GEM installs: 
gem install --source http://repo-st.oak.iparadigms.com/gem_repo
Or you can download them directly and install locally. Most of our Gems are sourced from rubygems.org and cached by bundler, but we do keep some here in case.

Unstable RPMs for Testing

In order to use this repository you will need to create the file /etc/yum.repos.d/sye_unstable.repo with the following contents: 

[sye_unstable]
name=Unstable RPMs for Centos $releasever - $basearch
baseurl=http://repo-st.oak.iparadigms.com/repos/sye/$releasever/unstable/
enabled=1
gpgcheck=0

Custom RPMs

In order to use this repository you will need to create the file /etc/yum.repos.d/sye.repo with the following contents: 

[sye]
name=System Team Custom RPMs for Centos $releasever - $basearch
baseurl=http://repo-st.oak.iparadigms.com/repos/sye/$releasever/stable/
enabled=1
gpgcheck=0

CentOS Package Repository

In order to use this repository you will need to create the file /etc/yum.repos.d/centosdvd.repo with the following contents: 

[centosdvd]
name=CentOS Enterprise Linux Distribution DVD - x86_64
baseurl=http://repo-st.oak.iparadigms.com/repos/centos/$releasever/repo/
enabled=1
gpgcheck=1
gpgkey=file:///etc/pki/rpm-gpg/RPM-GPG-KEY-CentOS-5
exclude=postgresql*
