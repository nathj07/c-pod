Systems Team Repository

This server hosts all things used in configuring our OS and application environments:
  • Downloads for various things
  • Installation Images for the operating systems we use
  • Kickstart Configurations
  • GEM Repository
  • YUM Repositories
    • Stable RPMs with our customizations
    • Unstable RPMs under testing
    • CentOS Distribution for network installs
Note that our current standard build of Centos is 5.9 but 6.4 is being tested now...

Downloads

There are other files that are useful. For example the latest copy of VMware tools

Installation Images (ISOs)

These image files are used by the Kickstart installation process. For example:
  • CentOS-5.9-x86_64-netinstall.iso - A minimal size boot image (~15Mb) that allows you to start a networked kickstart install.

    Type linux ks=http:///downloads/centos5-vm.ks?host=aname.local to begin.

  • CentOS-6.4-x86_64-netinstall.iso - A minimal size boot image (~240Mb) that allows you to start a networked kickstart install.

    Type linux ks=http:///downloads/centos6-vm.ks?host=aname.local to begin.

Note: the 'minimal' CentOS 6.4 image is hardly small at 240Mb! It would be nice to find a smaller one.

These images are also all mounted for browsing

GEM Repository

We have created some of our own GEMS, so a GEM repository has been setup. To use this add the source option to your GEM installs: 
gem install --source http:///gem_repo
Or you can download them directly and install locally.

YUM Repositories

Custom RPMs (Stable)

In order to use this repository you will need to create the file /etc/yum.repos.d/custom.repo with the following contents: 

[custom]
name=Custom RPMs for CentOS $releasever - $basearch
baseurl=http:///yum_repos/custom/$releasever/stable/
enabled=1
gpgcheck=0
Browse the Custom RPMs for: CentOS 5 and CentOS 6

Unstable Custom RPMs for Testing

When developing a new release of a custom RPM this repository is used. This is because YUM repos don't readily support multiple versions.

In order to use this repository you will need to create the file /etc/yum.repos.d/custom_unstable.repo with the following contents: 

[custom_unstable]
name=Unstable Custom RPMs for CentOS $releasever - $basearch
baseurl=http:///yum_repos/custom/$releasever/unstable/
enabled=1
gpgcheck=0
Browse the Unstable Custom RPMs for: CentOS 5 or CentOS 6

CentOS Distribution Package Repositories

To save bandwidth and speed-up networked installs the Distribution repositories are available. In order to use them create the file /etc/yum.repos.d/centosdvd.repo with the following contents: 

[centos_repo]
name=CentOS $releasever Linux Distribution DVD - x86_64
baseurl=http:///osdisks/centos$releasever
enabled=1
gpgcheck=1
gpgkey=http:///RPM-GPG-KEY-CentOS-$releasever

Browse the Distribution DVD Repository for: CentOS 5 and CentOS 6
